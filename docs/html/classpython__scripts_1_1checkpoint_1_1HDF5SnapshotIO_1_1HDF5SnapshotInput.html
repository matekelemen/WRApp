Detailed Description

Input class for reading most data from an HDF5 snapshot to a model part.

@classname HDF5SnapshotInput

Data read:

  • nodal solution step data
  • nodal data value
  • nodal flag
  • element data value
  • element flag
  • condition data value
  • condition flag
  • process info
